Javascript Objects

What is a Javascript Object?

A JS Object in its simplest form can be understood as a digital representation of a real-world object. To use a different example from W3Schools, I would liken it to a dog.

A dog has attributes like hair color, weight, breed, name, age, and more. These attributes are known in JS as properties.
A dog also has behaviors like sit, run, bark, sleep, eat, jump, fetch, and many more. These behaviors are known as methods.

All dogs will have these properties and methods, but the values of each property and the execution of each method will vary from dog to dog.

Javascript


// This is a how to set a prototype for an object called 'dog'. This is called a constructor function.

function dog(name, breed, age, haircolor) {
    this.name = name;
    this.breed = breed;
    this.age = age;
    this.hairColor = haircolor;
    this.dogYears = function() {return this.age * 7;};
}

// Using the 'new' keyword, objects of the type 'dog' can be created.

var myDog = new dog("Freckles", "Dalmation", 7, "white with black spots");
var notMyDog = new dog("Fifi", "Poodle", 3, "white");

// Display the values in an HTML element.

document.getElementById("demo").innerHTML = "This is my dog, " + myDog.name +
    ". He is a " + myDog.age + " year old " + myDog.breed + ". As you can see, his fur is " +
    myDog.hairColor + ". He is " + myDog.dogYears() + " years old in Dog Years.";
