CSS3 Transformations

CSS3 Transforms take a block-level HTML element and perform some action on it instantly. The basic actions that can be performed (not including 3D transforms for now) are: Rotate, Scale, Skew, Translate, and Matrix (which combines several). These transformations can be limited to either the X or Y axes, or, in most case be combined into one. Rotate is the only one whose non X or Y version, actually rotates on the Z axis.

Provided Code Sample

CSS


#demo div {
    width: 300px;
    height: 100px;
    background-color: darkkhaki;
    border: 1px solid black;
}
#demo2 {
    background-color: lightslategray;
    transform: scale(.5,.5);
}
#demo1 {
    background-color: aquamarine;
    transform: rotate(25deg);
}
#demo3 {
    background-color: beige;
    transform: translate(400px,-175px);
}

#rotate:hover {
    transform:rotate(40deg);
}
#rotateX:hover {
    transform:rotateX(155deg);
}
#rotateY:hover {
    transform:rotateY(170deg);
}
#scale:hover {
    transform:scale(1.25,.75);
}
#scaleX:hover {
    transform:scaleX(1.8);
}
#scaleY:hover {
    transform:scaleY(.25);
}
#skew:hover {
    transform:skew(20deg,30deg);
}
#skewX:hover {
    transform:skewX(30deg);
}
#skewY:hover {
    transform:skewY(125deg);
}
#translate:hover {
    transform:translate(50px,-40px);
}
#translateX:hover {
    transform:translateX(-30px);
}
#translateY:hover {
    transform:translateY(70px);
}
#matrix1:hover {
    transform:matrix(-1, 0.1, -0.3, -.5, 20, 10);
}

    transform:matrix(.5, -0.5, 0.2, -1, -50, 30);
}
#matrix3:hover {
    transform:matrix(1, 0, 0.5, 1, 45, -50);
}
